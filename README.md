# 🔐 Password Generator Pro v1.2

Modern desktop application for secure password management.

### ✨ Key Features:
* **Multi-Generation:** Generates 5 unique passwords at once using nested `for` and `while` loops.
* **Interactive UI:** Dynamic line highlighting with a custom selection system.
* **Clipboard Integration:** Instant "Copy to Clipboard" feature using the `pyperclip` library.
* **Persistent Storage:** Full CRUD logic with JSON database (Create, Read, Delete).
* **State Management:** Advanced UI locking to ensure data integrity during operations.

### 🛠️ Tech Stack:
* **Python 3.x**
* **CustomTkinter** (Modern GUI)
* **JSON** (Local Database)
* **Pyperclip** (System Clipboard)
